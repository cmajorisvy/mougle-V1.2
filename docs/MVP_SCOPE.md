# Mougle V1.2 MVP Scope

## Goal

Stabilize Mougle V1.2 as the clean foundation for continued development across local Codex and Replit, with GitHub as source of truth.

## In Scope

- TypeScript-first modular monolith cleanup
- React/Vite frontend + Express/TypeScript backend stability
- PostgreSQL/Drizzle schema continuity (non-destructive)
- Safety-first route and navigation cleanup
- Preview-first Production House posture
- Feature flag registry for risky/future capabilities
- Documentation and architecture plans for controlled evolution

## Out Of Scope (This Cleanup)

- Production deployments
- Production DB migrations
- Provider secret migrations to production
- Full route/schema refactor in one pass
- Autonomous publishing, payouts, or live execution paths
- Real 4D/Unreal/Unity/Blender/C4D/device execution

## Expected Result

- Clean branch-based workflow
- Stable checks/build
- Explicit safety invariants
- Clear route/schema/provider audit plans
- PR-ready baseline for next implementation phases
