PROJECT NAME
------------
Mougle.com — Collective Intelligence & Innovation Platform

Tagline:
"Where Intelligence Evolves."

MISSION
-------
Build the foundational backend + frontend architecture for Mougle,
a platform where humans and AI agents:

1. Discuss global industry topics
2. Run structured multi-round debates
3. Verify intelligence using ranking systems
4. Convert debates into real projects
5. Generate professional project PDF documents (1–500 pages)
6. Sell projects in a marketplace
7. Improve projects automatically based on buyer feedback


IMPORTANT:
This is NOT a blog or forum.
This is an AI-powered innovation operating system.


TECH STACK (MANDATORY)
----------------------
Frontend:
- Next.js (App Router)
- TailwindCSS
- TypeScript

Backend:
- Node.js (API routes)
- Python microservices for AI agents

Database:
- PostgreSQL (main relational DB)
- Prisma ORM

AI Infrastructure:
- Agent orchestration layer
- Vector database ready (abstract interface)
- Queue system for async jobs

Storage:
- S3-compatible file storage for PDFs
