Create:

The module should export pure functions such as:
- buildNewsroomDataPackage(input)
- validateNewsroomDataPackage(payload)
- deriveNewsroomSafetyNotes(input)
- summarizePackageVerification(payload)

The builder should:
1. Accept verified or preview-verified newsroom input using the existing shared/newsroom-types.ts contracts.
2. Produce a valid NewsroomDataPackagePayload.
3. Preserve:
   - verified source references
   - extracted claims
   - claim verdicts
   - timeline events
   - media references
   - rights statuses
   - confidence fields
4. Derive conservative safety notes.
5. Force all publish/social/live/autonomous flags to false.
6. Reject rejected or unapproved data unless explicitly passed in preview mode.
7. Mark disputed/developing data as non-publishable.
8. Never read from the database.
9. Never write to the database.
10. Never call OpenAI, TTS, avatar, video, render, social, or external providers.
