Build a full-stack web application called Hybrid Intelligence Network (HIN) — a platform where Humans and AI Agents sign up, discuss topics, verify information, and receive reputation rankings based on factual accuracy and intelligence.

The system must support BOTH human users and autonomous AI agents as first-class participants.

CORE CONCEPT

The platform is a knowledge discussion network where:

• Humans provide real-world observations and expertise
• AI Agents provide analysis, verification, statistics, and reasoning
• AI evaluates claims for factual confidence
• Reputation and ranking are calculated dynamically
• Truth is weighted by evidence, not popularity

Users and agents can:

create discussions

post claims

reply and debate

attach evidence

receive AI verification scores

gain or lose reputation

TRUST ALGORITHM ENGINE

Implement a Trust Score calculator:

Truth Confidence Score (TCS):

TCS =
0.35 * EvidenceScore +
0.20 * ConsensusScore +
0.20 * HistoricalReliability +
0.15 * ReasoningScore +
0.10 * SourceCredibility

Each post automatically receives a TCS.

EvidenceScore:

weight evidence types (research, dataset, news, personal observation)

ConsensusScore:

agreement among independent agents

HistoricalReliability:

author’s past verified accuracy

ReasoningScore:

AI evaluation placeholder function

SourceCredibility:

normalized reputation score

Store TCS in database.

AI VOTING SYSTEM

Replace likes/upvotes with:

Multiple agents contribute independently.
