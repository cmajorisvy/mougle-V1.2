Next task: Add real OpenAI generation to Mougle AI Production House Prompt Studio, behind explicit safe-mode controls.

Goal:
Allow root-admin users to optionally use the OpenAI API to generate richer production plans, scripts, room descriptions, avatar profiles, camera plans, Unreal scene manifests, and 4D cue manifests.

Important:
Do not enable real Unreal sends.
Do not enable real 4D hardware sends.
Do not enable real provider calls by default.
Keep deterministic mock mode as the default.
Preserve all current tests and safety gates.

Requirements:

1. Add provider mode selector in Prompt Studio:
   - Mock deterministic mode
   - OpenAI generation mode

2. OpenAI mode must only be available if:
   - OPENAI_API_KEY exists
   - root-admin is authenticated
   - user explicitly selects OpenAI mode
   - user confirms generation
   - request is audit-logged

3. Never expose OPENAI_API_KEY or any secret value in API responses.

4. Add backend route:
   POST /api/admin/production-house/prompt-studio/generate-openai

5. The route should:
   - validate root-admin access
   - validate prompt input
   - call OpenAI only when OpenAI mode is explicitly requested
   - return structured JSON only
   - store the generated package through the ProductionHouseStorage adapter
   - append an audit log entry
   - preserve the locked SAFETY_ENVELOPE

6. The OpenAI output should produce:
   - production plan
   - news/podcast/hall/room script
   - room specification
   - avatar specification
   - camera shot list
   - Unreal scene manifest draft
   - 4D cue manifest draft
   - safety notes
   - approval status: Draft

7. Add strict schema validation for the OpenAI response.
   If the response is invalid, reject it safely and do not persist it as approved.

8. Add UI display:
   - “Generated by OpenAI” badge
   - token/cost placeholder if available
   - generated package preview
   - save as draft button
   - export JSON button

9. Preserve current mock mode:
   - deterministic prompt hash behavior must remain unchanged
   - existing prompt-studio tests must still pass

10. Add regression tests:
   - OpenAI mode is unavailable without OPENAI_API_KEY
   - mock mode still works without external API
   - OpenAI route requires root-admin
   - OpenAI route never leaks secrets
   - invalid OpenAI JSON is rejected
   - generated OpenAI package is saved as Draft only
   - no Unreal command is sent
   - no 4D command is sent
   - SAFETY_ENVELOPE remains unchanged

Do not add ElevenLabs, Meshy, Runway, Convai, NVIDIA ACE, Unreal real sends, or 4D real sends in this task.
