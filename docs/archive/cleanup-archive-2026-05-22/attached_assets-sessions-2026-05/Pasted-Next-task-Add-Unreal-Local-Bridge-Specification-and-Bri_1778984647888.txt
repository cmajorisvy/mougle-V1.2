Next task: Add Unreal Local Bridge Specification and Bridge Contract Viewer.

Goal:
Create a formal local bridge specification inside Mougle AI Production House that defines how a future external Unreal workstation bridge must receive, validate, and respond to production commands.

Important:
Do not enable real Unreal sends.
Do not connect to Unreal Engine.
Do not trigger Movie Render Queue.
Do not import assets into Unreal.
Do not send 4D hardware commands.
Do not publish anything.
This task is documentation, contract generation, validation schema, and UI only.

Scope:

1. Add new section:
Unreal Bridge Contract

This section should show:
- expected local bridge URL structure
- required authentication headers
- supported command types
- expected request payloads
- expected response payloads
- safety rules
- package validation requirements
- error response formats
- audit requirements

GET /api/admin/production-house/unreal/bridge-contract
