Build a new admin dashboard called:

Mougle AI Production House

This is not a simple news app. It is a production-house control dashboard for creating AI-powered 3D and 4D cinema-quality productions, Unreal Engine virtual rooms, active newsrooms, active podcast rooms, debate rooms, halls, interview rooms, AI avatars, and cinematic scene workflows using prompts.

The system should be designed as a control center that can generate structured production instructions, scene manifests, avatar manifests, 4D cue manifests, and Unreal Engine command payloads.

Important:
Replit is the AI production control center and admin dashboard.
Unreal Engine is the 3D/4D cinema renderer and virtual production engine.
External APIs generate voices, scripts, 3D assets, avatar data, motion data, and video assets.
A local bridge or Unreal Remote Control API handles Unreal Engine and 4D hardware communication.

Build the app with:
- React frontend
- Node.js / Express backend
- Clean modular architecture
- Admin dashboard layout
- API route structure
- Environment variables for all API keys
- JSON manifest generation
- No hard-coded secrets
- Production-ready folder structure
