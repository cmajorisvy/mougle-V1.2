10. Mathematical / algorithmic layer

Add a detailed section explaining the math / algorithms used or proposed.

A. News impact score

impactScore =
w1 * sourceCredibility
+ w2 * sourceCoverage
+ w3 * eventRecency
+ w4 * entityImportance
+ w5 * claimConfidence
+ w6 * socialVelocity
- w7 * disputeRisk
- w8 * legalRisk

Normalize 0–100.

B. Breaking news decision

isBreaking =
impactScore >= threshold
AND sourceCoverage >= minimumSources
AND legalRisk <= maxLegalRisk
AND humanVerificationStatus in allowedStates

C. Source coverage confidence

sourceCoverage =
uniqueIndependentSources / requiredSources

confidence =
0.4 * sourceCoverage
+ 0.3 * claimEvidenceScore
+ 0.2 * historicalSourceReliability
+ 0.1 * crossSourceAgreement

D. Cluster similarity

Jaccard(A,B) = |tokens(A) ∩ tokens(B)| / |tokens(A) ∪ tokens(B)|

cluster if:
Jaccard(titleA,titleB) >= threshold
AND timeDistance <= window

E. Claim confidence

claimConfidence =
evidenceStrength
* sourceReliability
* corroborationCountFactor
* freshnessFactor
* contradictionPenalty

mediaCandidateScore =
licenseSafetyWeight
+ relevanceWeight

Only choose candidates where:
