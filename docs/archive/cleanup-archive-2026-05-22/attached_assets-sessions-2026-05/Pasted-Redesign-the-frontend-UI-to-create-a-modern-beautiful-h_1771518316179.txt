Redesign the frontend UI to create a **modern, beautiful, highly user-friendly Human User Dashboard** for the Hybrid Intelligence Network platform.

The dashboard must support all existing features including discussions, AI News Updates, live debates, agents, reputation system, credits economy, and social interactions.

---

## DESIGN GOALS

* Clean modern SaaS-style interface
* Highly intuitive navigation
* Minimal cognitive overload
* Fast interaction flow
* Mobile responsive
* Professional AI media platform aesthetic

Design inspiration:
Linear.app, Notion, ChatGPT dashboard, Twitter/X modern layout.

Use:
React + TailwindCSS + component-based architecture.

---

## GLOBAL LAYOUT

Create a 3-zone layout:

LEFT SIDEBAR (Navigation)
CENTER CONTENT (Dynamic pages)
RIGHT SIDEBAR (Context widgets)

---

## LEFT SIDEBAR NAVIGATION

Include icons + labels:

🏠 Home
💬 Discussions
🧠 Live Debates
📰 AI News Updates
🤖 Agents
🏆 Rankings
💰 Credits Wallet
🔔 Notifications
👤 Profile
⚙️ Settings
(Admin only → Admin Panel)

Sidebar collapsible.
