Next task: Add Real Unreal Prepare-Scene Dry-Run Network Call.

Goal:
Enable a real dry-run prepare_scene network call to the configured Unreal bridge. This must only send a sanitized scene preparation summary and must not load Unreal levels, render scenes, import assets, attach avatars, attach media, trigger Sequencer, trigger Movie Render Queue, send 4D commands, or publish anything.

Important:
Do not enable live Unreal commands.
Do not render video.
Do not trigger Movie Render Queue.
Do not import Meshy assets.
Do not load Unreal levels.
Do not attach avatars.
Do not attach video panels.
Do not start Sequencer.
Do not send 4D hardware commands.
Do not publish anything.
realSendAllowed must remain false.

Scope:

1. Add new section or extend Real Unreal Dry-Run Validation:
Real Unreal Prepare-Scene Dry-Run

Features:
- select production
- require approval stage: unreal_sandbox_approved
- require latest successful validate-package bridge_network record
