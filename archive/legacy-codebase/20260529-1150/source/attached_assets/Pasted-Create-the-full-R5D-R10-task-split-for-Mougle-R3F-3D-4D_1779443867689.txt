R5E — Deterministic GLB/GLTF validator only

Goal:
Create a deterministic server-side GLB/GLTF validator.

Scope:

* Add server/services/gltf-validator.ts
* Add focused tests for validator
* No DB usage
* No object storage writes
* No HTTP routes
* No provider calls
* No renderer/headless browser

Validator must check:

* GLB magic 0x46546C67
* version 2
* total length matches buffer length
* JSON chunk type 0x4E4F534A
* JSON parses
* optional BIN chunk type 0x004E4942
* asset.version === "2.0"
* node/mesh/accessor/bufferView caps
* byteSize <= 25 MB default
* reject unsupported extensionsRequired
* reject external image URIs
* allow only data URI or bufferView images if needed
* return { ok: true, metadata } or { ok: false, reason }

Create:
docs/reports/R5E_GLTF_VALIDATOR_REPORT.md

Verification:

* tests pass
* no DB change
* no route change
* no object storage writes
