4. Safety statuses

Define enums and allowed transitions:

licenseStatus:
- unknown
- pending_review
- licensed
- internal_only
- rejected
- expired

publicSafetyStatus:
- private_only
- admin_preview_only
- safe_for_internal_review
- safe_for_public_preview
- blocked

approvalStatus:
- draft
- needs_review
- approved_internal
- approved_public
- rejected
- archived
- quarantined

renderMode:
- preview_only
- dry_run
- approved_render

Default values must be safest:
- publicSafetyStatus = admin_preview_only or private_only
- approvalStatus = draft
- renderMode = preview_only
- publicSafePreview = false
- publicUrl = null
- signedUrl = null
- realSendAllowed = false
- executionEnabled = false

5. State machine

Add Mermaid stateDiagram-v2 for asset approval:

uploaded/private
→ metadata_extracted
→ license_review
→ safety_review
→ internal_preview
→ approval_board
→ approved_internal
→ approved_public

Blocked paths:
- rejected
- quarantined
- archived

Only approved_public may allow publicUrl.
Only approved_render may allow executionEnabled.
Never allow realSendAllowed without explicit approval.

6. R3F safety gate rules

Design validation functions:

canPreviewAssetInAdmin(asset)
canUseAssetInScene(asset, scenePackage)
canExposeAssetPublicly(asset)
canRenderScenePackage(scenePackage)
canUseUnityBuild(scenePackage)
canUseAvatarRig(asset)
canUseProviderGeneratedAsset(asset)

Each must check:
- license status
- public safety status
- approval status
- object path type
- publicUrl/signedUrl state
- renderMode
- executionEnabled
- realSendAllowed

7. Object storage policy

Design object paths:

Private:
PRIVATE_OBJECT_DIR/production-assets/{assetId}/original
PRIVATE_OBJECT_DIR/production-assets/{assetId}/preview
PRIVATE_OBJECT_DIR/scene-packages/{packageId}/manifest.json

Public only after approval:
PUBLIC_OBJECT_SEARCH_PATHS/production-assets/{assetId}/public-preview

Rules:
- private assets use signed admin URLs only
- publicUrl remains null until approval
- no client-side provider credentials
- no direct bucket browsing
- no public asset before approved_public
